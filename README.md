# AI Chat

A Flutter mobile app with an AI-powered chat backend built on Supabase Edge Functions and OpenAI.

## Project Structure

```
AI-Chat/
├── client/          # Flutter mobile app
└── server/          # Supabase backend
```

- [client/](client/README.md) — Flutter app (Dart, flutter_bloc, Supabase Flutter SDK)
- [server/](server/README.md) — Supabase Edge Functions (Deno/TypeScript, OpenAI)

## Stack

- **Client:** Flutter (Dart), flutter_bloc, Supabase Flutter SDK
- **Server:** Supabase Edge Functions (Deno/TypeScript), OpenAI API
- **Auth:** Sign in with Apple, Supabase Auth

## Prerequisites

- [Flutter](https://flutter.dev/docs/get-started/install) SDK
- [Supabase CLI](https://supabase.com/docs/guides/cli)
- An [OpenAI API key](https://platform.openai.com/api-keys)

See each subdirectory's README for detailed setup instructions.
